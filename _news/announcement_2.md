I developed a pet project - an agent-based multimodal RAG. This project is a chatbot that answers questions about the Beber annual report for 2023, modifying the user query if necessary and adding the results found by the Tavily Search tool to the context. The idea of the solution is taken from the article ["Corrective Retrieval Augmented Generation"](https://arxiv.org/pdf/2401.15884) and adapted for use in this context.
